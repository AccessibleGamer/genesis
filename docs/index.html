Hello Operator. I exist to assist you with finding and tracking information for your journey in Warframe.
To get started using my tracking features, ask me to start with /start.
In order to see my configuration options, ask me to /track.
If you'd like to see all of my options, ask for /help.
All other commands are available in /help.
Join Cephalon Genesis' server!

How to Use

 Genesis' alias is either / or @genesis
In the room that you want to track alerts, call /start.
If you also want to have a seperate news room, /untrack news in your alerts room. Then /start in your news room, /untrack all and /track news
There are many different tracking options available, and they are all available from /track
Switch platforms using the /platform <Platform>. Valid platforms are PC, PS4, and X1.

Features

 Track Warframe data, get notifications about alerts, sorties, etc.
